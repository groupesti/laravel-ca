# Support

## Getting Help

### Bug Reports

If you have found a bug, please open an issue on [GitHub Issues](https://github.com/groupesti/laravel-ca/issues) using the [bug report template](.github/ISSUE_TEMPLATE/bug_report.md).

When filing a bug report, always include:

- Package version (`composer show groupesti/laravel-ca`)
- PHP version (must be 8.4+)
- Laravel version (12.x)
- A minimal code snippet or repository that reproduces the issue

### Feature Requests

Feature requests are welcome. Please use the [feature request template](.github/ISSUE_TEMPLATE/feature_request.md) on GitHub Issues.

### Questions and Discussions

For general questions about usage, architecture, or best practices, please use [GitHub Discussions](https://github.com/groupesti/laravel-ca/discussions) instead of opening an issue.

GitHub Issues are reserved for **confirmed bugs** and **actionable feature requests** only.

## What We Do Not Support

- PHP versions below 8.4
- Laravel versions below 12.x
- Deployment or server configuration assistance
- Third-party package integration issues unrelated to this package

## Security Vulnerabilities

If you discover a security vulnerability, please **do not** open a public issue. See [SECURITY.md](SECURITY.md) for reporting instructions.
